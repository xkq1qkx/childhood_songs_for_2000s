# 🎵 童年影视歌曲网站

一个展示童年影视歌曲的静态网站，支持表格和画廊两种视图模式。

## 功能特性

- 📊 **表格视图**：完整展示所有歌曲信息
- 🖼️ **画廊视图**：卡片式展示，更加美观
- 🔍 **搜索功能**：快速查找歌曲、作品或歌手
- 🏷️ **筛选功能**：按印象程度和类型筛选
- 📱 **响应式设计**：完美适配手机和电脑
- 🔗 **直接跳转**：点击即可跳转到B站观看

## 部署到 GitHub Pages

1. 在 GitHub 创建新仓库（例如：`childhood-songs`）
2. 将所有文件上传到仓库
3. 进入仓库设置 Settings → Pages
4. Source 选择 `main` 分支
5. 点击 Save，等待几分钟即可访问

访问地址：`https://你的用户名.github.io/仓库名/`

## 本地预览

直接双击打开 `index.html` 文件即可在浏览器中预览。

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- 无需任何框架或构建工具

## 数据更新

如需更新歌曲数据，编辑 `data.js` 文件即可。
